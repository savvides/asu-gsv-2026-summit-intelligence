# Contributing

This is a knowledge resource from the ASU+GSV 2026 Summit. Contributions that improve accuracy or add context are welcome.

## What's useful

- **Corrections**: Speaker names, affiliations, or quotes that need fixing in the transcripts
- **Context**: Additional session details, missing metadata, or links to related resources
- **Analysis**: Thematic connections across sessions, or additional demand signals from the summit

## How to contribute

1. Open an issue describing the change
2. If submitting a pull request, keep changes focused and explain what they improve

## Review

Submissions are reviewed by [Philippos Savvides](mailto:philippos.savvides@asuep.org) within two weeks.
